# Calculator
基于编译原理所学词法分析、语法分析、语义计算等知识制作而成的简约计算器。

该项目与[@VeraRyan](https://github.com/VeraRyan)共同完成。（其实是我们的实验作业）

文件结构说明：

	- Calculator - 总界面
	- LexDFAFrame - 词法分析器界面
	- LexDFA - 词法分析实现
	- SyntaxFrame - 语法分析器界面
	- Syntax - 语法分析实现
	- CalcFrame - 计算器界面
	- Calc - 含语义计算的计算器实现
